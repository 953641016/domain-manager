## 1. 获取已创建飞书应用的凭证

既然您已经在飞书开放平台创建好应用和机器人了，请按以下步骤获取凭证：

### 1.1 登录飞书开放平台
1. 访问 https://open.feishu.cn/
2. 登录您的企业管理员账号

### 1.2 进入应用详情页
1. 进入"控制台" → "应用管理"
2. 找到您刚才创建的"域名管家"应用
3. 点击进入应用详情页面

### 1.3 获取应用凭证（App ID 和 App Secret）
在应用详情页的左侧菜单找到 **"凭证与基础信息"**（或者"凭证"）：

| 配置项 | 位置 | 说明 |
|--------|------|------|
| **App ID** | 凭证与基础信息 → App ID | 形如 `cli_xxxxxxxxxxxxxxxx` |
| **App Secret** | 凭证与基础信息 → App Secret | 点击"显示"或"重置"获取 |
| **Verification Token** | 凭证与基础信息 → 事件订阅 → Verification Token | 用于验证飞书请求来源 |
| **Encrypt Key** | 凭证与基础信息 → 事件订阅 → Encrypt Key | (可选) 用于事件加密 |

⚠️ **重要提示：**
- App Secret 只显示一次，请妥善保存
- 如果忘记了，可以点击"重置"重新获取（但会使旧的失效）

### 1.4 配置重定向URL（OAuth登录需要）
在应用详情页左侧找到 **"安全设置"**（或"OAuth配置"）：
1. 找到 **重定向URL** 配置项
2. 添加您的系统域名：
   ```
   https://d.fwxg.com/api/v1/auth/feishu/callback
   ```
   *(如果本地调试，也可以添加 `http://localhost:3000`)*

### 1.5 配置IP白名单（安全防护）
在应用详情页左侧找到 **"安全设置"**：
1. 找到 **IP白名单** 配置项
2. 添加您的服务器IP：
   ```
   115.28.211.155
   ```
3. 保存配置

⚠️ **重要：** 启用IP白名单后，只有白名单内的服务器才能调用飞书API，提高安全性。

## 2. 获取您自己的飞书用户ID

### 方式一：通过飞书开放平台获取（推荐）
1. 进入飞书开放平台 → 应用详情页
2. 在左侧找到"成员与权限"或者"权限管理"
3. 查看应用可见范围内的用户列表
4. 找到您自己，复制 `user_id`（通常是 `ou_` 开头的）

### 方式二：通过飞书机器人获取
1. 向您的飞书机器人发送任意消息
2. 后续系统能获取到您的用户ID（需要先部署系统）

### 方式三：通过飞书通讯录
1. 打开飞书电脑版
2. 进入通讯录找到自己
3. 点击头像或详情，可能能看到用户ID

⚠️ **注意：** 飞书用户ID通常是 `ou_` 开头的一串字符，请确认您获取的是正确的 `user_id`，而不是 `union_id` 或 `open_id`。

## 3. 配置环境变量

### 3.1 编辑环境变量文件
在服务器或本地的项目中，编辑 `backend/.env` 文件：

```ini
# ------------------------------
# 飞书应用配置
# ------------------------------
FEISHU_APP_ID=cli_xxxxxxxxxxxxxxxx
FEISHU_APP_SECRET=xxxxxxxxxxxxxxxxxxxxxxxx
FEISHU_VERIFICATION_TOKEN=xxxxxxxxxxxxxxxx
FEISHU_ENCRYPT_KEY=xxxxxxxxxxxxxxxxxxxxxxxx
SUPER_ADMIN_FEISHU_USER_ID=ou_您的飞书用户ID
```

### 3.2 环境变量说明
| 变量名 | 必须？ | 说明 |
|--------|--------|------|
| `FEISHU_APP_ID` | ✅ 是 | 飞书应用的 App ID |
| `FEISHU_APP_SECRET` | ✅ 是 | 飞书应用的 App Secret |
| `FEISHU_VERIFICATION_TOKEN` | ✅ 是 | 事件订阅的 Verification Token |
| `FEISHU_ENCRYPT_KEY` | 否 | 事件加密的 Key（可选）|
| `SUPER_ADMIN_FEISHU_USER_ID` | ✅ 是 | 超级管理员的飞书用户ID |

## 4. 初始化超级管理员

### 4.1 运行数据库初始化脚本

部署到服务器后，运行以下命令：

```bash
cd /path/to/domain-manager/backend

# 激活虚拟环境（如果使用）
source venv/bin/activate

# 运行初始化脚本
python scripts/init_db.py
```

### 4.2 初始化脚本会做什么？
- ✅ 创建所有需要的数据表
- ✅ 自动创建超级管理员用户
- ✅ 将您配置的 `SUPER_ADMIN_FEISHU_USER_ID` 绑定到超级管理员
- ✅ 如果已存在，会更新飞书用户ID

### 4.3 验证初始化成功
运行完初始化脚本后，您应该看到类似输出：
```

============================================================

1. 创建数据表...
   ✓ 数据表创建完成

2. 创建默认超级管理员用户...
   ✓ 超级管理员用户创建成功
   - ID: 1
   - 姓名: 超级管理员
   - 飞书ID: ou_1234567890abcdef
   - 角色: super_admin

...
============================================================
数据库初始化完成!
============================================================
```

## 5. 首次登录

### 5.1 使用飞书扫码登录
1. 访问 `https://d.fwxg.com`
2. 点击"飞书登录"按钮
3. 使用飞书APP扫码登录
4. 系统会自动识别您是超级管理员

### 5.2 如果登录遇到问题
- 检查飞书应用的重定向URL是否正确配置
- 确认您的用户ID是否正确填入 `SUPER_ADMIN_FEISHU_USER_ID`
- 检查飞书应用的权限范围是否包含"获取用户基本信息"

---

## 6. 后续配置机器人Webhook（可选）

部署完成后，如果需要机器人功能：
1. 在飞书开放平台 → 应用详情 → 事件订阅
2. 配置请求地址：`https://d.fwxg.com/api/v1/feishu/webhook`
3. 选择需要订阅的事件（如"接收消息"等）
